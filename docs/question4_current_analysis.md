# Question 4 Analysis

最后更新：2026-04-20

本文基于当前主模型结果目录 [`results/runs/model_current`](../results/runs/model_current) 对题目第 4 问进行整理：

> 分析各机型使用情况，并提出是否需要进一步调整机队或航班计划的建议。

本说明不再把第 4 问理解为“再做一个优化模型”，而是把它作为基于现有机型分配结果的管理分析模块来完成。

---

## 1. 当前方案的总体画像

基于 [`scenario_summary.csv`](../results/runs/model_current/scenario_summary.csv)：

- 总收入：`11,891,210.82`
- 总成本：`6,426,543.33`
- 总利润：`5,464,667.49`
- 总未满足需求：`13,686.6`
- 平均载客率：`86.0%`
- 加权载客率：`87.48%`
- 高影子价格航段数：`549`

这说明当前方案总体盈利能力较强，但网络中仍有大量航段处于容量紧张状态。第 4 问的重点因此不应是“是否还能求出一个解”，而应是：

- 哪些机型是当前真正的主力资源
- 哪些航班/航段最紧张、最值得追加容量
- 哪些配置明显偏大、可以降级或重审
- 哪些机型当前并不值得扩充

## 4. 调整建议分析

### 4.1 优先建议扩容或换大机型

当前最明确的建议方向是：对高负载、高利润、正影子价格的任务优先扩容。

优先级最高的对象包括：

- `TFB <-> BOD`
- `TFB <-> GBJ`
- `TFB <-> EDB`
- `TFB -> IOM`
- `IOM -> BOD`
- `PHH -> QEY`

这类任务已经体现出：

- 运力约束真实存在
- 增加容量有直接收益空间
- 当前核心承运机型正是 `F16C0Y165` 或 `F0C0Y76`

因此，若有微调空间，最优先的动作不是“全面改排班”，而是优先把这些任务上的机型向更高运力方向调整。

### 4.2 可以考虑降级机型的任务

被标记为 `Downgauge` 的任务共有 `95` 个，典型例子包括：

- `AA0092` `BOD -> TFB`，载客率 `68.73%`
- `super_AA0232`，载客率 `68.67%`
- `BA2265` `ZWS -> TFB`，载客率 `68.85%`
- `super_BA2167`，载客率 `55.64%`
- `BA2266` `TFB -> ZWS`，载客率 `57.09%`
- `AA0128` `GBJ -> TFB`，载客率 `63.39%`

这些任务的共同特征是：

- 载客率中低
- 影子价格接近 0
- 仍有正利润，但容量紧张度不高

因此，这些对象不是“应该取消”，而是“可以优先考虑换小机型，以释放更大机型去支撑高紧张航线”。

### 4.3 应重点复核时刻或保留必要性的任务

被标记为 `Review_Schedule` 的任务虽然只有 `6` 个，但管理意义很强：

- `AA0138` `BOD -> PSE`，载客率 `7.11%`，利润 `-13,890.73`
- `super_BA2214` `TFB -> MXT`，载客率 `13.52%`，利润 `-2,081.3`
- `BA2484` `TFB -> CMJ`，载客率 `18.33%`，利润 `-480.8`
- `AA0082` `BOD -> TFB`，载客率 `49.94%`，利润 `-218.8`
- `super_BA2321` `IMO -> TFB`，载客率 `27.5%`，利润 `-138.1`
- `BA2111` `QTD -> TFB`，载客率 `26.32%`，利润 `-84.3`

这些任务的含义不是“立刻删除”，而是：

- 至少不应继续按当前口径机械保留
- 需要结合网络连接价值进一步确认
- 若缺少显著连接支撑，应优先进入时刻/班次复核名单

### 4.4 对机队调整的管理建议

综合机型层和任务层的结果，可以形成较稳定的建议：

1. 若需要扩充运力，优先考虑 `F16C0Y165` 与 `F0C0Y76`
2. 不建议优先扩充当前完全闲置的机型家族
3. 应优先通过“降级低紧张任务”来释放主力机型，而不是简单增加总机队
4. 对极低载客率且亏损的任务，应进入班次/时刻审查清单

## 5. 任务四的最终回答口径

如果直接面向题目作答，推荐这样组织：

1. 先说明并购后机队使用明显分化：
   - `F16C0Y165` 与 `F0C0Y76` 是主力且偏紧张
   - 多个机型家族明显闲置

2. 再说明网络中存在大量容量瓶颈：
   - 高影子价格航段达到 `549`
   - 多条高利润航班满载运行

3. 然后指出最值得保留和扩容的高价值任务/航段：
   - 重点集中在 `TFB-BOD`、`TFB-GBJ`、`TFB-EDB`、`TFB-IOM` 等

4. 最后给出调整建议：
   - 对高价值满载任务优先换大机型或补运力
   - 对中低负载、低影子价格任务优先考虑降级
   - 对极低载客率且亏损任务进行时刻/航班复核
   - 机队扩张若要发生，应优先围绕当前主力机型，而非全面扩张
