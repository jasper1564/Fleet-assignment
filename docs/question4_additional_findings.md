# 问题四补充发现

最后更新：2026-04-23

## 1. 关于“为什么有些机型完全没有被使用”

当前结果里，确实存在部分机型 `0` 使用的情况，例如：

- `F8C12Y126`
- `F12C30Y117`
- `F8C12Y99`
- `F12C0Y132`
- `F12C0Y112`

这个现象看起来反常，但结合当前模型结构后，更合理的解释并不是“模型没有找到最优解”，而是：

> 在当前建模口径下，这些机型在成本与座位配置上的综合性价比较低，因此没有进入最优分配方案。

原因主要有两点：

1. 当前模型对机队数量采用的是“最多可用多少架”的约束，而不是“必须把所有飞机都派出去飞”的约束。
2. 当前目标函数是“期望收入 - 飞行成本”，因此如果某类机型无法带来更高的净收益，模型会理性地让它闲置。

## 2. 这一发现的业务含义

这说明当前结果更像是“机型结构错配”，而不是“总机队规模不足”。

也就是说：

- `F16C0Y165` 和 `F0C0Y76` 之所以被大量使用，是因为它们在当前网络里兼具较强的适配性和较好的成本效率。
- 部分 0 使用机型之所以被闲置，更可能是因为它们在“座位规模 - 每小时飞行成本 - 当前任务结构”这个组合下不占优。

因此，“有飞机没用上”本身不能直接证明模型没找到最优解；相反，它更像是在提醒我们，当前网络真正短缺的是高适配主力机型，而不是总飞机数量。

## 3. 初步证据

从参考机队成本表中按“每座每小时成本”粗看：

- `F16C0Y165`：约 `27.89`
- `F0C0Y76`：约 `31.45`
- `F0C0Y72`：约 `33.19`
- `F12C12Y48`：约 `39.75`
- `F12C30Y117`：约 `44.09`
- `F8C12Y99`：约 `46.43`
- `F12C0Y132`：约 `47.58`
- `F8C12Y126`：约 `66.27`

可以看到，当前最核心的主力机型恰好也是单位座位成本更有优势的机型，而多个完全闲置机型的单位座位成本明显更高。

这进一步支持了下面这句判断：

> 当前闲置现象更像是性价比筛选结果，而不是求解失败。

## 4. 对后续模型解释的建议

在问题四的表述里，建议把这个发现写成：

> 当前部分机型未被使用，并不必然意味着模型失效。结合模型的目标函数和机队成本结构，更合理的解释是：在既定航班计划和现有参数下，这些机型的单位运力成本偏高、性价比较低，因此未进入最优分配方案。
